## Run Backend
Install Express, modules necessary to run the app: 
```
$ npm i express 
```
Install Nodemon, modules necessary to run the app:
```
$ npm i -g nodemon
```
Install jsonwebtoken, modules necessary to run the app:
```
$ npm i jsonwebtoken
```
Run the back:
```
$ npm run devstart
```
